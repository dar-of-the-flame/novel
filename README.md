НАБЛЮДАТЕЛЬ - Текстовая психологическая драма
Игра в жанре текстового квеста с элементами визуальной новеллы и философского экзистенциализма.

Особенности
Атмосферный пролог с философским повествованием
Система выбора, влияющая на развитие сюжета
Достижения за различные пути прохождения
Кинематографические эффекты и переходы
Адаптивный дизайн для мобильных устройств
Управление
Клик/тап - продолжить диалог или сделать выбор
Пробел/Enter - альтернативное управление
Кнопка звука - включить/выключить музыку
Технологии
Чистый JavaScript без фреймворков
CSS анимации и эффекты
HTML5 Audio для музыки
LocalStorage для сохранения прогресса
